My Personal Portfolio Website

##  Project Description
This is my personal portfolio website built using **HTML5** and **CSS3**.  
The website is designed with a clean and simple layout to showcase my profile and provide contact information.

It consists of two main pages:
-  Home Page  
-  Contact Page  

This project helped me practice structuring web pages with HTML and styling them using CSS.

---

##  Pages Included

###  Home Page
- Introduction section  
- Basic information about me  
- Simple and clean design  
- Navigation to Contact page  

###  Contact Page
- Contact information  
- Simple contact form (if included)  
- Navigation back to Home page  

---

##  Features
- Multi-page website  
- Simple and responsive layout  
- Easy navigation between pages  
- Built using only HTML and CSS  

---

##  Technologies Used
- HTML5  
- CSS3  

---

##  Purpose of the Project
The purpose of this project is to practice front-end web development fundamentals and create a basic personal portfolio website.

---

##  How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in your browser
